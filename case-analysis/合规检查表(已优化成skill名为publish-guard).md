**合规检查表**

适用范围：Q版2D虚拟人物、头像/壁纸、评论互动、粉丝投稿、生活照片二创

**使用原则：我只发布真实、原创、可授权、可解释的内容；凡涉及交易、健康、未成年人、他人肖像、AI生成和粉丝投稿，发布前必须二次检查。**

## 一、失败案例与风险结论

| **失败类型**    | **案例/依据**                                                                                     | **我的规避动作**                                           |
|-----------------|---------------------------------------------------------------------------------------------------|------------------------------------------------------------|
| 虚假种草/代写   | 小红书曾治理虚假内容，并起诉虚假种草通告平台；公开资料提到封禁品牌/商家、删除虚假评价、处置账号。 | 不能买量、代写、伪造用户反馈或假装真实体验。               |
| 过度滤镜误导    | 旅行/景点内容因过度美化引发信任危机；问题本质是让用户把创作图误认为真实参考。                     | 生活照二创、AI图、滤镜图必须避免误导真实场景。             |
| 炫富/虚假人设   | 平台曾治理炫富与伪造创业人设。                                                                    | 虚拟IP不能包装成真实富有人设、真实成功经历或夸张消费能力。 |
| 黑医美/烟草软文 | 历史争议显示医美、烟草、药品类软文风险高。                                                        | 本账号不碰医美、药品、烟草、减肥功效等高危赛道。           |
| AI与版权风险    | 生成式AI规则要求尊重知识产权、肖像权、隐私权，并对图片/视频等生成内容按规定标识。                 | AI图、二创图、用户照片必须确认授权和标识边界。             |

## 二、发布前合规检查表

| **检查项**         | **禁用/高风险内容**                                                                                       | **允许替代表达**                                                         |
|--------------------|-----------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------|
| 赛道敏感词         | 医美、减肥、药品、烟草、酒精、博彩、投资、贷款、玄学改命、成人擦边、未成年人恋爱/性暗示、政治灾难蹭热点。 | 只做可爱、陪伴、头像、壁纸、评论互动；涉及健康/金钱/未成年人时直接回避。 |
| 绝对化词           | 最、第一、唯一、永久、100%、必火、稳赚、无风险、官方认证、全网首个、保证涨粉。                            | 改成“我觉得”“可能适合”“尝试版”“个人设定”“非官方”。                       |
| 不能承诺的效果     | 治好焦虑/抑郁、保证开心、陪你走出低谷、一周涨粉、必爆、包变现、用了就有效。                               | 表达为情绪陪伴和审美体验：希望给你一点陪伴感/轻松感。                    |
| 不能虚构的经历     | 假装真实用户投稿、假评论、假订单、假合作、假获奖、假注册版权、假装角色真实到过某地。                      | 明确“角色设定/二创/粉丝授权投稿”；数据、合作、版权只写已确认事实。       |
| 不能使用的视觉素材 | 明星照片、影视截图、品牌Logo、三丽鸥/迪士尼等未授权IP、他人插画、未授权粉丝照片、未标识AI深度合成。       | 使用自有角色资产、可商用素材、已授权投稿；保留授权记录。                 |
| 评论互动边界       | 诱导私信交易、加微信、群二维码、索要隐私、承诺每条必回、让粉丝发敏感照片。                                | 评论区只做公开互动；不收集手机号、地址、身份证、未成年人隐私。           |
| 商业合作边界       | 软广不标注、伪装自发推荐、夸大产品效果、接医美/药品/烟草/金融类推广。                                     | 商业内容显著标注合作；早期不接高风险品类。                               |
| AI内容边界         | AI生成真人脸、仿冒他人画风、误导成真实拍摄、生成暴力/色情/歧视内容。                                      | AI仅辅助草图/背景；角色主视觉保持原创；必要时标注AI辅助。                |

## 三、我的发布前流程

| **步骤** | **我检查什么**                                     | **处理方式**                 |
|----------|----------------------------------------------------|------------------------------|
| 1\. 选题 | 是否涉及健康、金钱、未成年人、政治灾难、成人擦边？ | 有则放弃或重写。             |
| 2\. 文案 | 是否有绝对化、功效承诺、虚构经历、诱导私信？       | 改成主观感受和创作说明。     |
| 3\. 图片 | 是否用了未授权IP/真人/品牌/粉丝照片/AI深度合成？   | 确认授权、替换素材或标注。   |
| 4\. 发布 | 是否说明角色虚构、投稿授权、商业合作？             | 必要时在正文或评论置顶说明。 |
