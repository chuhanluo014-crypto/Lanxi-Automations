# 小红书样本标注规范 v1

## 1. 目的

把 `xhs_sample_pool_v1.csv` 中的原始笔记，转成可用于蓝汐内容设计、样本召回和复盘的结构化参考。

- 标注单位：一行样本对应一条小红书笔记。
- 标注依据：标题、正文、标签、封面和图片内容。信息不足时填“无”或选择最保守的项，不猜测作者未表达的设定。
- 标注范围：当前先试标 30 条；规则稳定后再覆盖全部 190 条。
- 字段填写：多选项使用英文分号 `;` 分隔，最多填 2 项；不使用自由发挥的同义词。

## 2. 标注总原则

1. 先看图，再看标题、正文和标签；视觉信息与文字冲突时，在 `notes` 写明原因。
2. 填“这条笔记实际在做什么”，不是“你觉得它应该做什么”。
3. 主题、场景、情绪、钩子各选最主要的 1-2 项；不要把所有可能项都填上。
4. `risk_level` 评价内容风险，`keep_status` 评价是否值得作为蓝汐的参考，两者不能混用。
5. 参考不等于模仿。保留的是内容结构、互动方法、情绪和视觉方向，不复制角色、文案或具体构图。

## 3. 字段规范

### `topic`：内容主题

必填，选 1 项；确有双主题时最多 2 项。

| 可选值 | 适用情况 |
|---|---|
| `IP诞生/角色介绍` | 介绍角色名字、设定、世界、能力或首次出现。 |
| `头像/壁纸分享` | 头像、手机壁纸、Apple ID 头像、社交主页素材。 |
| `日常陪伴` | 角色日常、轻情绪、陪伴感、治愈型小片段。 |
| `粉丝共创` | 征集名字、画粉丝 ID、评论区点单、to 签。 |
| `表情包/社交素材` | 聊天表情包、微信三件套、朋友圈背景、桌面组件。 |
| `节日/节点内容` | 生日、节日、换季、考试、毕业、上班等时间节点。 |
| `角色设定/创作过程` | 人设拆解、设计过程、画法、制作过程、形象迭代。 |
| `潮玩/周边` | 玩偶、拼豆、盲盒、周边、商品化表达。 |
| `梗图/热点二创` | 热点、游戏、影视、网络梗、反差搞笑。 |
| `其他` | 不属于以上类别，必须在 `notes` 说明。 |

### `scene`：内容使用或观看场景

必填，选 1-2 项。

| 可选值 | 适用情况 |
|---|---|
| `社交主页/头像` | 用作微信、小红书、Apple ID 等头像或主页素材。 |
| `手机/桌面壁纸` | 手机壁纸、锁屏、桌面、组件。 |
| `聊天互动` | 表情包、回复、to 签、评论区互动。 |
| `睡前/独处` | 睡前、发呆、独处、安静陪伴。 |
| `学习/工作` | 上班、考试、学习、疲惫、打工人情绪。 |
| `节日/纪念日` | 节日、生日、毕业、周年、换季。 |
| `朋友/关系互动` | 情头、闺蜜、crush、朋友互动。 |
| `日常娱乐` | 日常分享、玩梗、游戏、消遣。 |
| `无明确场景` | 只有角色展示或场景无法判断。 |

### `emotion`：主要情绪

必填，选 1-2 项。

| 可选值 | 判断标准 |
|---|---|
| `治愈` | 安抚、陪伴、柔和、被理解。 |
| `可爱` | 萌感、软乎、角色吸引力。 |
| `安静/松弛` | 发呆、轻缓、低刺激、休息感。 |
| `好奇` | 初来乍到、探索、提问、发现。 |
| `搞怪` | 抽象、玩梗、反差、故意不正经。 |
| `鼓励` | 加油、考试、成长、祝福、打气。 |
| `亲近/友情` | 朋友、陪伴、闺蜜、共同记忆。 |
| `低落/孤独` | 委屈、疲惫、没人理解、轻微丧感。 |
| `浪漫` | crush、情侣、心动、礼物感。 |
| `无明确情绪` | 单纯展示素材或信息不足。 |

### `hook`：吸引点击、收藏或评论的主要钩子

必填，选 1-2 项。

| 可选值 | 判断标准 |
|---|---|
| `角色初见` | 新 IP、角色名字、角色设定或首次露面。 |
| `头像可用` | 明确可领取、可做头像、可保存。 |
| `壁纸/素材可用` | 壁纸、组件、背景、模板等实用素材。 |
| `情绪共鸣` | 读者能代入疲惫、孤独、可爱、日常情绪。 |
| `视觉反差` | 强记忆视觉、反差感、特殊画风或角色动作。 |
| `评论区征集` | 征集名字、需求、点单、投票、留言。 |
| `粉丝被记住` | 画 ID、to 签、抽选、定制回应。 |
| `节日/热点` | 节日、考试、毕业、游戏、影视、热点。 |
| `故事悬念` | 角色背景、连续剧情、问题或未完待续。 |
| `无明确钩子` | 纯展示且无明显互动或传播设计。 |

### `reusable_structure`：可复用内容结构

必填，选 1 项。用“内容结构”描述，不记录具体角色或原句。

| 可选值 | 结构说明 |
|---|---|
| `角色诞生+互动邀请` | 名字/设定出现 + 简短情绪介绍 + 低压力留言邀请。 |
| `头像素材+领取提示` | 多张可用图片 + 使用场景 + 保存/领取提示。 |
| `日常情绪+陪伴短句` | 一个轻情绪画面 + 一两句陪伴表达。 |
| `粉丝征集+创作回馈` | 征集名字/需求 + 承诺画图或反馈。 |
| `节日祝福+角色回应` | 节日节点 + 角色祝福 + 可收藏画面。 |
| `角色设定+小故事` | 设定说明 + 一个小事件或背景片段。 |
| `热点借势+角色反应` | 热点/梗 + 角色化反应或二创。 |
| `制作过程+结果展示` | 过程、迭代、制作细节 + 成品。 |
| `纯视觉展示` | 主要依赖图片，无明显文字结构。 |

### `risk_level`：内容风险

必填，选 1 项。

| 可选值 | 判断标准 |
|---|---|
| `低` | 原创表达、常规陪伴/头像/壁纸/轻互动，无明显争议。 |
| `中` | 涉及热点、影视游戏二创、暗示商业授权、夸张承诺、边缘情绪内容，使用前需人工改写。 |
| `高` | 明显搬运、侵权风险、攻击真实人物、外貌焦虑、虚假承诺、医疗金融、暴力血腥、强争议内容。 |

### `keep_status`：是否保留为蓝汐参考

必填，选 1 项。

| 可选值 | 判断标准 |
|---|---|
| `保留` | 与蓝汐的安静、温柔、陪伴、头像/壁纸、粉丝共创方向匹配，且可借鉴结构或视觉方法。 |
| `降权` | 有局部可借鉴点，但整体气质不适合蓝汐，例如高能量搞怪、强热点依赖或过度商业表达。 |
| `剔除` | 风险高、信息质量差、图片不可用、明显不相关，或无法作为原创参考。 |

### `notes`：标注依据和补充说明

选填，但 `降权`、`剔除`、`中`、`高` 风险样本必须填写。

建议格式：`依据：……；可借鉴：……；注意：……`。

示例：`依据：评论区征集名字并承诺画头像；可借鉴：低压力互动入口；注意：角色气质偏高能量，蓝汐仅参考互动结构。`

## 4. 标注流程

1. 打开样本的标题、正文、封面和图片。
2. 依次填写 `topic`、`scene`、`emotion`、`hook`、`reusable_structure`。
3. 判断 `risk_level` 和 `keep_status`。
4. 对不确定、降权或剔除的样本写 `notes`。
5. 每标完 10 条，检查是否出现大量“其他”或“无明确”；若超过 20%，先更新本规范再继续。

## 5. 蓝汐首发帖示例

假设样本是一条“新 IP 蓝汐诞生，邀请读者留下名字”的首发帖：

| 字段 | 标注值 |
|---|---|
| `topic` | `IP诞生/角色介绍` |
| `scene` | `社交主页/头像;聊天互动` |
| `emotion` | `治愈;安静/松弛` |
| `hook` | `角色初见;评论区征集` |
| `reusable_structure` | `角色诞生+互动邀请` |
| `risk_level` | `低` |
| `keep_status` | `保留` |
| `notes` | `依据：介绍角色名字、陪伴感和后续画名字的计划；可借鉴：低压力留言邀请。` |

## 6. 试标验收

30 条试标完成后，检查：

- 每条必填字段是否都有值。
- `其他`、`无明确` 是否过多。
- 同类样本是否被标成不同类别。
- `降权`、`剔除` 是否写明原因。
- 是否能从标注中筛出“适合蓝汐”的头像/壁纸、陪伴、粉丝共创和首发参考样本。
